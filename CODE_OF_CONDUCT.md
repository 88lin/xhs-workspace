# Code of Conduct

## Our Standard

This project expects contributors to be:

- Respectful in technical disagreement
- Clear about tradeoffs and risks
- Careful with user data, creator content, and attribution
- Constructive in reviews and issue discussions

## Unacceptable Behavior

- Personal attacks, harassment, or intimidation
- Bad-faith review behavior or deliberate obstruction
- Posting private credentials, private datasets, or sensitive creator materials
- Misrepresenting third-party licensing or authorship

## Enforcement

Project maintainers may edit, hide, lock, or remove content that violates this code of conduct. Repeated or severe violations may lead to temporary or permanent exclusion from the project spaces.
